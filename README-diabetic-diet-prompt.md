# Practical Diabetic Diet Planner (Prompt Engineering Project)

This project demonstrates how **AI Prompt Engineering** can be used to generate **realistic meal plans for diabetic patients** using structured prompting.

The prompt is designed using the **RTCFR framework**, which helps guide AI models to produce high-quality, practical outputs.

The goal of this project is to generate a **7-day diabetic-friendly meal schedule for South Indian vegetarian households** that:

- Avoids eggs
- Uses familiar foods
- Requires minimal cooking
- Works for busy working families
- Helps maintain stable blood sugar levels

---

# RTCFR Prompt Framework

RTCFR stands for:

**R — Role**  
Defines the expertise the AI should assume.

**T — Task**  
Clearly defines what the AI must generate.

**C — Context**  
Provides situational constraints and real-life conditions.

**F — Few-Shot Examples**  
Examples that guide the model toward correct outputs.

**R — Report Format / Tone**  
Defines output structure and tone.

This structured prompting significantly improves response quality.

---

# Problem Being Solved

Many diabetic diet plans online are unrealistic because they:

- Require extensive cooking
- Include uncommon ingredients
- Ignore busy work schedules
- Do not align with local food habits

This prompt ensures that generated meal plans are:

- **Culturally appropriate**
- **Quick to prepare**
- **Affordable**
- **Sustainable for everyday life**

---

# Diet Constraints Used

The AI is instructed to generate meal plans with the following constraints:

- South Indian vegetarian diet
- Strictly **no eggs**
- Focus on **low glycemic foods**
- Limit white rice portions
- Avoid sugar and processed foods
- Include high fiber foods and legumes
